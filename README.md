# 🎨 BthLab Designs Sandbox

معرض تصاميم تجريبي للفعاليات — **مستقل تماماً** عن مشروع tiktok-live-reader و BthLab الأساسي.

## الغرض
استعراض تصاميم بصرية مقترحة قبل اعتمادها وتطبيقها على المشروع الحقيقي.

## التشغيل محلياً
```bash
npm install
npm start
```
ثم افتح `http://localhost:3000`

## النشر على Railway
1. أنشئ ريبو جديد على GitHub (مثلاً: `bthlab-designs-sandbox`)
2. ارفع الملفات
3. اربط الريبو بـ Railway → سيتم النشر تلقائياً

## التصاميم المتوفرة

### الفعاليات
- 🎡 **العجلة**: نيون / كازينو / 3D / ديناميكية بصور
- ⚔️ **حرب الكلمات**: ساحة معركة بـ HP
- 🐎 **سباق الخيل**: مضمار ليلي
- 🧊 **الجليد**: مكعبات تتشقق وتتهشم
- 🏰 **حرب القلاع**: قذائف بمسار قوسي
- 💀 **الخروج**: حلبة / شجرة بطولة / موت مفاجئ
- 🔤 **خمن الكلمة**: بطاقات تنقلب / كشف تدريجي / مطر تخمينات
- 📝 **تسجيل المشاهدين**: فقاعات / مدرجات / يانصيب / cyberpunk

### لوحة الأدمن
- 🟦 **Bento Modern**: شبكة bento عصرية
- 📋 **Sidebar Pro**: sidebar كلاسيكي احترافي
- ✨ **Glass Hub**: glassmorphism بخلفية متحركة
- ⚡ **Compact Pro**: مدمج وعملي
